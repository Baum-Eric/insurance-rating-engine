# insurance-rating-engine
Actuarial project building a frequency-severity model for insurance pricing using R
# Overview
Developed a motor insurance rating engine using a frequency–severity modeling approach to estimate expected premiums based on policyholder risk characteristics.

## Methods
- Frequency modeled using Poisson regression  
- Severity modeled using Gamma regression  
- Data split into training (70%) and validation (30%) sets  
- Premium calculated as: Frequency × Severity  

## Results
- Models captured key relationships between risk factors and claims  
- Evaluation performed using RMSE on validation data  
- Results showed reasonable predictive performance  

## Tools Used
- R (glm, statistical modeling)  
- Data visualization and analysis  

## Key Concepts
Actuarial modeling, frequency-severity models, regression analysis, insurance pricing
